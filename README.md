Instalasi
Klon Repository:
https://github.com/SatrioUtomo13/auth-system.git

Instal Dependensi:
Untuk backend (Node.js):
cd backend
npm install

Untuk frontend (React.js):
cd frontend
npm install

Menjalankan Aplikasi
Backend:
cd backend
node index

Frontend:
Buka terminal di direktori frontend.
Jalankan perintah:
cd frontend
npm start
